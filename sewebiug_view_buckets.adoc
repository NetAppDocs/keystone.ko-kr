= 버킷을 봅니다
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Bucket 목록에는 선택한 서브테넌트의 기존 버킷을 표시합니다.

. 목록을 보려면 메뉴에서 * 오브젝트 스토리지 > 버킷 * 을 선택합니다.
. 필요한 경우 서브테넌트 드롭다운 메뉴를 사용하여 다른 서브테넌트를 선택합니다.
+
다음과 같은 간단한 정보가 각 버킷에 표시됩니다.

+
** 이름
** 지역
** 크기
** 생성 날짜
** 오브젝트
** 작업(예: 버킷 삭제)

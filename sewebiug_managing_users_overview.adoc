= 개요
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
이 섹션에서는 사용자를 관리하는 방법에 대해 설명합니다.


NOTE: 사용자를 관리하려면 고객 관리자 액세스 권한이 필요합니다.

* 사용자 목록을 봅니다.
* 사용자를 추가합니다.
* 이름, 성, 표시 이름, 사용자가 액세스할 수 있는 기간, 해당 기간 내의 사용자 역할 등의 사용자 세부 정보를 수정합니다.
* 사용자를 삭제합니다.

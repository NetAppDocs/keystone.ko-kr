= 클라우드 서비스를 삭제합니다
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
이 방법을 사용하여 NetApp 서비스 엔진 웹 포털에서 CVS 구독을 삭제할 수 있습니다.

.단계
. 메뉴에서 * 클라우드 서비스 * > * 클라우드 서비스 * 를 선택합니다.
. 목록에서 클라우드 서비스를 찾고 삭제 아이콘을 클릭합니다.
. 삭제 확인 대화 상자에서 삭제할 클라우드 서비스의 이름을 입력합니다.
. 확인 * 을 클릭합니다.

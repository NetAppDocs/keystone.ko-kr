= 사용자 목록을 봅니다
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
사용자 목록에는 로그인한 사용자가 액세스할 수 있는 모든 사용자의 목록이 표시됩니다. 목록을 보려면 메뉴에서 관리 > 사용자 를 선택합니다.

목록에는 사용자 이름, 이름 및 성, 전자 메일 주소, 기본 테넌시, 만든 날짜 등 각 사용자에 대한 간단한 정보가 표시됩니다.

각 사용자 옆에 있는 작업 아이콘을 사용하여 사용자를 수정하거나 삭제할 수 있습니다.
